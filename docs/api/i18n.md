# i18n（多語系）

v3 內建 `en` / `zh-TW` / `zh-CN` 三套語系，用於頁尾與空資料等內建字串。

## `DataTableLocale` 型別

```ts
interface DataTableLocale {
  emptyMessage: string            // 空資料提示，如 'No Available Data'
  rowsPerPageMessage: string      // 每頁筆數標籤，如 'rows per page:'
  rowsOfPageSeparatorMessage: string  // 分頁資訊分隔字，如 "1–5 of 12" 的 'of'
}
```

## 選擇內建語系

```vue
<DataTable :headers="headers" :items="items" locale="zh-TW" />
```

`locale`：`'en'`（預設）| `'zh-TW'` | `'zh-CN'`。

| locale | emptyMessage | rowsPerPageMessage | separator |
| --- | --- | --- | --- |
| `en` | No Available Data | rows per page: | of |
| `zh-TW` | 無可用資料 | 每頁筆數： | / |
| `zh-CN` | 无可用数据 | 每页条数： | / |

## 覆寫個別字串 / 自訂語系

`localeOverrides` 可覆寫部分字串，或傳入**完整** `DataTableLocale` 以支援未內建的語言：

```vue
<!-- 只覆寫部分 -->
<DataTable ... locale="en" :locale-overrides="{ emptyMessage: 'Nothing here' }" />

<!-- 完整自訂語系（例如日文） -->
<DataTable ... :locale-overrides="{
  emptyMessage: 'データがありません',
  rowsPerPageMessage: '1ページあたりの行数：',
  rowsOfPageSeparatorMessage: '/',
}" />
```

## 解析優先序

字串依下列順序解析（高者覆蓋低者）：

```
個別 message prop（emptyMessage / rowsPerPageMessage / rowsOfPageSeparatorMessage）
  ＞ localeOverrides
    ＞ 內建 locale 包
```

> 既有的個別 message props 仍可用且**優先序最高**（向後相容）。`en` 預設值與舊版相同，原英文使用者無感。

## 直接取用內建語系包

```ts
import { locales } from '@tiaohsun/vue-datatable-tailwind'
console.log(locales['zh-TW'].emptyMessage) // '無可用資料'
```
